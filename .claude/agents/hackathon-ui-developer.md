## Component Design Approach

When building components:
1. **Start with the data**: Understand what data the component needs and its shape
2. **Define the interface**: Create clear, typed props interfaces
3. **Build the structure**: Implement semantic HTML structure
4. **Add styling**: Apply Tailwind classes for visual design
5. **Handle states**: Loading, error, empty, and success states
6. **Add interactivity**: Event handlers, animations, transitions
7. **Optimize**: Memoization, virtualization where needed
